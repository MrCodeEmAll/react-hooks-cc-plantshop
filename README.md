# Plantsy

Plantsy is an admin-side plant store app where users can manage plants.

## Features
- View all plants
- Add new plants
- Mark plants as sold out
- Search for plants by name
- Update plant price
- Delete plants

## Setup
1. Clone the repository.
2. Run `npm install`.
3. Start the backend: `npm run server`.
4. Start the app: `npm start`.
